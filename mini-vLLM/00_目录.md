## 与 vLLM 对照速查

| nano-vllm | vLLM v1 | 差异 |
|-----------|---------|------|
| `llm.py` | `AsyncLLM` | 无 IPC，同进程 |
| `engine/llm_engine.py` | `EngineCore` | 合并了 EngineCore + EngineCoreProc |
| `engine/scheduler.py` | `Scheduler` | 更简单，无 DP/PP |
| `engine/block_manager.py` | `KVCacheManager` + `block_pool.py` | 合二为一 |
| `engine/model_runner.py` | `GPUModelRunner` | 有 CUDA Graph 支持 |
| `engine/sequence.py` | `Request` | 更精简 |
| `layers/attention.py` | PagedAttention kernel | Python 实现，非 CUDA kernel |
